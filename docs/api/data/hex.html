Hexedecimal API (bd103.data.hex)¶

Provides methods and classes to translate various data types to and from the hex format.

This module is heavily based off of other data format modules, like json and pickle.

See also

  • Hexadecimal on Wikipedia

class bd103.data.hex.HexDecoder(list_item_len: Optional[int] = None)[source]¶

Used for turning a string of hexedecimals into an integer or list of integers.

Parameters

list_item_len – The size of each chunk of data used in decode_list().

decode_int(o: str) → int[source]¶

Decodes a hex string into one integer.

This function converts each character then multiplies it by 16 if there is another character after it.

Parameters

o – A string to be converted into an integer. Some examples are ff, e8, or 00.

Note

This method even decodes uncommon forms of hexedecimal, which means that any string you give has a chance of being larger than 255. If you want to limit the maximum value, take a look at decode_list().

decode_list(o: str) → list[int][source]¶

Decodes a hex string into a list of integers.

The size of each integer returned is determined by list_item_len. If it is equal to 2, then decode_list() would parse fe3a as [254, 58].

Parameters

o – A valid hexedecimal string.

encode_list(o: list[int]) → str[source]¶

Encodes a list of integers into one string.

Parameters

o – A list of integers to be encoded.

Note

Make sure you know the maximum size of each integer. If you wish to decode the values again, you will need to determine the value of HexDecoder.list_item_len. If any integer is greater than 15, it will have to have a chunk size of 2. Greater than 255 deems a chunk size of 3.
